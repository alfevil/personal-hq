# Personal HQ Telegram Bot

Здесь находится серверная часть для Telegram-бота, которая отвечает на команду `/start` красивым приветственным сообщением и кнопкой для запуска мини-приложения (Mini App).

## Как запустить

1. Установи зависимости:
   ```bash
   npm install
   ```

2. Создай файл `.env` в этой папке и добавь туда:
   ```
   TELEGRAM_BOT_TOKEN=твой_токен_от_botfather
   WEBAPP_URL=ссылка_на_твой_развернутый_react_frontend
   ```

3. Запусти бота:
   ```bash
   npm start
   ```
